The Kendall distance is a measure of similarity between two rankings. The distance is 0 if the rankings are identical, and the higher the value, the more dissimilar the rankings are.

Kendall distance is used in some voting aggregation algorithms (i.e. Kemeny Consensus), which aim to find a final ranking with minimal Kendall distance to each voters ranking.

That is, for each pair of candidates, count 1 if the candidates are not in the same order in the 2 rankings. The total after counting each of these disagreements is the Kendall distance.
