The Borda Count is a voting system in which candidates gain points based on their position in each voters' ranking. The candidate that accumulates the most points when totalled across all rankings is the winner. The system intends to choose candidates that are acceptable to a broad number of people rather than to find one that is preferred by a majority. For each ranking, assign to each candidate a number of points equal to the number of candidates that are ranked lower.
